# HopperHacks 2025

## Getting Started

Follow these steps to set up and run the project:

1. **Clone the repository:**
   ```sh
   git clone <repo-url>
   ```

2. **Navigate to the project directory:**
   ```sh
   cd HopperHacks2025
   ```

3. **Install dependencies:**
   ```sh
   npm install
   ```

4. **Run the development server:**
   ```sh
   npm run dev
   ```
